処理画面

　処理中のとき、処理の進行状況を表示する画面を表示します。

　

ウィンドウ

　

処理内容 (Caption)

　処理の機能内容を表示します。

　

処理ファイル (File, Total Process)

　現在処理中のファイル名と、ファイルリスト内の進行状況を表示します。

　

ファイルの処理状況 (Current Phase)

　現在処理中のファイルの進行状況を表示します。

　

中止 (Cancel)

　処理を強制終了します。

　

　

結果画面

　処理終了後、処理結果のログを表示します。

　

保存 (Save)

　処理結果テキストをファイルに保存します。

　

コピー (Copy)

　処理結果テキストをクリップボードにコピーします。

　

閉じる (Close)

　ウィンドウを閉じます。
