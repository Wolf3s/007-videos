help コマンド

目的

　プログラムのバージョン情報、ヘルプ情報を表示します。

　

書式

command 部

　help (省略形：h)

　

options 部

オプション 意味
-v バージョン情報のみを表示します。

　

target file 部

なし

　

　パラメータとしてコマンド名を付加すると、そのコマンドのヘルプメッセージを表示します。
