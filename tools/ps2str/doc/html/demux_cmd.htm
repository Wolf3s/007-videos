demux コマンド

目的

　多重化されているストリームの非多重化を行います。

　

書式

command 部

　demux (省略形：d)

　

options 部

オプション 意味
-o 出力ファイルが既に存在した場合、上書き出力します。
-d [ディレクトリ] 出力ディレクトリを指定します。
-v 作業の進行状況を詳しくレポートします。
-video(:番号) (番号：0 - 15) MPEG ビデオストリームのみを出力します。番号 (省略可能) を指定した場合は、そのストリーム番号のデータのみが出力されます。
-audio(:番号) (番号：0 - 31) MPEG オーディオストリームのみを出力します。番号 (省略可能) を指定した場合は、そのストリーム番号のデータのみが出力されます。
-ipu(:番号) (番号：0 - 65535) ipu 動画ストリームのみを出力します。番号 (省略可能) を指定した場合は、そのストリーム番号のデータのみが出力されます。
-pcm(:番号) (番号：0 - 65535) PCM オーディオストリームのみを出力します。番号 (省略可能) を指定した場合は、そのストリーム番号のデータのみが出力されます。
-adpcm(:番号) (番号：0 - 65535) adpcm オーディオストリームのみを出力します。番号 (省略可能) を指定した場合は、そのストリーム番号のデータのみが出力されます。
-data(:番号) (番号：0 - 65535) 任意データストリームのみを出力します。番号 (省略可能) を指定した場合は、そのストリーム番号のデータのみが出力されます。

　出力ストリームを指定しなかった場合、すべてのストリームが出力されます。

　

target file 部

パラメータ１ 非多重化される pss ファイル (MPEG2 プログラムストリームファイル)

　出力されるファイル名は、次の規約によって自動的に生成されます。

ストリーム ファイル名 (xxx は、入力ファイル名。id は、ストリーム番号)
MPEG ビデオ xxx_video_id.m2v
MPEG オーディオ xxx_audio_id.m2a
ipu 動画 xxx_ipu_id.ipu
pcm オーディオ xxx_pcm_id.ads
adpcm オーディオ xxx_adpcm_id.ads
data xxx_data_id.dat
