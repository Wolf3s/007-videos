convert コマンド

目的

　MPEG 動画から IPU 動画への変換を行います。

　

書式

command 部

　convert (省略形：c)

　

options 部

オプション 意味
-o 出力ファイルが既に存在した場合、上書き出力します。
-d [ディレクトリ] 出力ディレクトリを指定します。
-v 作業の進行状況を詳しくレポートします。
-pss(:ストリーム番号) 入力ファイルを MPEG2 プログラムストリームとして処理します。(ストリーム番号を省略した場合、最初に発見される、MPEG ビデオストリームが処理対象になります。)
-video (デフォルト) 入力ファイルを MPEG ビデオエレメンタリストリームとして処理します。

　

target file 部

パラメータ１ 変換元となる MPEG ビデオファイル
パラメータ２ (省略可能) 出力ファイル (省略された場合は、変換元ファイルの拡張子を、ipu としたファイルに出力されます。)
